## 🏢 Corporate Profile // نبذة عن المؤسسة
**NexusAutomata AI** هو نظام إدراكي متطور مصمم خصيصاً لأتمتة العمليات التشغيلية للشركات والمطورين العقاريين الكبار (Automated Broker Agencies). يقوم النظام برصد المكالمات الفائتة 24/7 وتحويلها فوراً إلى محادثات ذكية عبر الواتساب والـ SMS للإجابة على الأسعار، المساحات، وجدولة المعاينات تلقائياً.

---

## 🗺️ Repository Architecture // هيكل المنظومة التقني
المستودع يحتوي على 5 ملفات برمجية مترابطة تشكل الهيكل الكامل للموقع:

1. `index.html` - المنصة الرئيسية السينمائية (Core Terminal) وتضم الجرافيكس المتحرك وصندوق التعريف بالشركة وباقات الأسعار.
2. `register.html` - صفحة التقاط بيانات الشركات وتأكيد مواصفات الباقة المختارة تلقائياً.
3. `checkout.html` - بوابة الدفع المؤمنة (InstaPay / Bank Interconnect).
4. `success.html` - صفحة تأكيد نجاح الاشتراك وتفعيل العقل الرقمي للعميل.
5. `demo.html` - نظام المحادثة التجريبي الحي (Simulated WhatsApp Interface) لتجربة الـ AI قبل الشراء.
